# MIPS-Verilog
Implementation of various digital circuits of the MIPS Processor in Verilog

## Contents
### Basic Gates 
+ AND Gate
+ OR Gate
+ NAND Gate
+ NOR Gate
+ EXOR Gate
+ EXNOR Gate
+ NOT Gate

### Combinational Circuits
+ Demux
+ Encoder
+ Decoder

## Sequential Circuits
+ SR Flip Flop
+ D Flip Flop
+ 4-bit UpDown Counter

## Multi Cycle Pipelined Implementation
+ Add Instruction
